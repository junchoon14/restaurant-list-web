1. **可以在首頁看到所有餐廳與它們的簡單資料：**
- [ ] 餐廳照片
- [ ] 餐廳名稱
- [ ] 餐廳分類
- [ ] 餐廳評分
  
2. **可以再點進去看餐廳的詳細資訊：**
- [ ] 類別
- [ ] 地址
- [ ] 電話
- [ ] 描述
- [ ] 圖片
 
3. **可以透過搜尋餐廳名稱來找到特定的餐廳**
